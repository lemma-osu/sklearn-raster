# Installation

## From Source

!!! info
    `sknnr-spatial` will be released on PyPI soon. Until then, you can install the package from source.

```bash
pip install git+https://github.com/lemma-osu/sknnr-spatial
```

## Dependencies

- Python >= 3.9
- scikit-learn >= 1.2
- numpy
- dask
- typing-extensions
